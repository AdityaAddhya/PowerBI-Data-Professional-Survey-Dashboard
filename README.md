# PowerBI-Data-Professional-Survey-Dashboard
This project presents an interactive Power BI dashboard analysing survey data of data professionals. It highlights salary trends, job satisfaction, work-life balance, programming language preferences, and career entry challenges, transforming raw data into meaningful insights for industry trend analysis.
This project presents an interactive Power BI dashboard built using a survey dataset of data professionals across different roles, industries, and experience levels. The visualization aims to uncover key insights into salary trends, job satisfaction, work-life balance, preferred programming languages, and career entry challenges in the data industry.
The dashboard transforms raw survey data into meaningful, data-driven insights to help understand workforce patterns in the analytics and data science ecosystem.
- Objectives
  - Analyze salary distribution across various data roles
  - Understand job satisfaction and work-life balance trends
  - Identify popular programming languages among data professionals
  - Examine difficulty levels in entering the data field

- Tools & Technologies Used
  - Power BI – Data visualization and dashboard development
  - Microsoft Excel – Data preprocessing and cleaning
  - DAX (Data Analysis Expressions) – Calculated measures and metrics
- Dataset Information
  The dataset consists of survey responses from data professionals worldwide. It includes information such as:
  - Job Role
  - Salary Range
  - Programming Language Preference
  - Country of Employment
  - Work-Life Balance Rating
  - Career Satisfaction Score
  - Entry Difficulty into Data Field

- Key Insights Generated
  - Data Scientist and Data Engineer are the Highest paid Job roles.
  - Python dominates as the favourite programming language among data professionals
  - 75% of Data Professionals are Male.
